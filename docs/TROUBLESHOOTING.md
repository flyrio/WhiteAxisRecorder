# 常见问题

## 没有导出文件
- 确认 `启用记录` 已开启
- 确认 `自动导出` 已开启，或在 `战斗记录` 中点击 `导出选中`
- 自动导出仅在副本完成 / 团灭 / 离开副本时触发
- 检查 `自定义路径` 是否填写正确，可点击 `打开目录` 直接进入
- 默认输出路径:`D:\Triggerlines`

## 队伍减益没有记录
- 确认勾选 `记录队伍减益(自己/队友)`
- 仅记录自己/队友身上的减益
- 同一目标+状态在去重间隔内只记录一次
- 濒死/衰弱/受伤加重/伤害降低不会记录

## Boss增益没有记录
- 确认勾选 `记录Boss增益(Boss身上)`
- 仅记录Boss目标身上的增益
- 导出时目标选择器为周围敌人，队友/自身不会匹配
- 同一目标+状态在去重间隔内只记录一次
- 濒死/衰弱/受伤加重/伤害降低不会记录

## 平A没有记录
- 确认勾选 `记录平A`
- 仅记录敌人的平A

## 技能效果没有记录
- 确认勾选 `记录效果触发`
- 仅记录敌人的技能效果

## 状态条目太多/重复
- 状态类条目按目标+状态在去重间隔内合并（状态名/ID双重去重）
- 可调整 `状态去重间隔(ms)` 来平衡刷屏与遗漏

## 读条条目太多/重复
- 同技能读条在 300ms 内合并为一条

## 战斗记录列表为空
- 确认已发生过副本完成 / 团灭 / 离开副本
- 确认 `启用记录` 已开启

## AbilityEffect 重复
- AbilityEffect 默认 400ms 去重

## 触发条件不符合预期
- 导出条件节点默认勾选高亮显示并写入颜色，若界面未显示请重新导入文件
- 导出结构为 `TreeRoot -> TreeParallel -> TreeSequence -> TreeCondNode`，每条记录一个序列节点属于正常现象
- 状态剩余时间单位为毫秒（ms），请检查 LeftTime 与比较条件
- 请参考 `docs/EXPORT_FORMAT.md`
- 可在触发树中手动调整
