# 使用说明

## 基本流程
1. 在 AEAssist 中启用插件
2. 进入副本自动开始记录(可在 UI 中关闭)
3. 战斗结束保存本把记录到列表，可在列表中编辑并导出

## 界面选项
- `启用记录`: 自动记录开关
- `记录队伍减益(自己/队友)`: 记录自己/队友身上的减益与持续时间
- `记录Boss增益(Boss身上)`: 记录 Boss 身上的增益与持续时间
- `状态去重间隔(ms)`: 状态类条目去重窗口（0 表示不去重，默认 300ms）
- `记录效果触发`: 记录敌人技能效果（AbilityEffect，默认关闭）
- `记录平A`: 记录敌人平A（默认关闭）
- `自动导出`: 战斗结束自动导出开关
- `立即导出`: 立即导出当前记录
- `清空`: 清空当前记录
- `自定义路径`: 手动填写导出目录（留空则使用默认路径，支持相对路径）
- `保存`: 保存自定义路径
- `打开目录`: 打开当前生效的导出目录
- `条目颜色`: 可自定义各类条目的颜色（读条/效果/平A/减益/Boss增益）
- `战斗记录`: 查看每一把的记录列表，可选择导出/删除
- 以上设置会保存到 `AEAssist根目录\Settings\WhiteAxisRecorder.json`
## 状态记录规则
- 队伍减益与 Boss 增益按目标+状态去重（窗口内重复不会记录）
- 若持续时间显著回满或层数变化，会再次记录
- 状态剩余时间以毫秒（ms）记录与导出
- Boss 增益导出时目标选择器为周围敌人
- 濒死/衰弱/受伤加重/伤害降低不会记录
- 同技能读条在 300ms 内合并为一条


## 自动导出
- 导出条件节点默认勾选高亮显示，颜色来自条目颜色预设
- 导出结构:`TreeRoot -> TreeParallel -> TreeSequence -> TreeCondNode`，每条记录对应一个序列节点
- 勾选 `自动导出` 时，副本完成 / 团灭 / 离开副本会在保存到列表后自动导出
- 关闭 `自动导出` 时，战斗结束只保存到列表，不会导出
- 手动关闭记录功能时不会自动导出

## 战斗记录
- 战斗结束后会保存为一条记录，不会立刻删除
- 可在列表中选择记录，勾选保留条目并点击 `导出选中`
- 可使用 `全选 / 全不选 / 删除未选 / 删除该记录` 管理记录
- 状态类条目(减益/增益)按目标+状态去重

## 导出位置
- 留空时默认导出到 `D:\Triggerlines`
- 自定义路径支持相对路径（以 AEAssist 根目录为基准）
- 文件名稳定，包含地形ID、职业ID与时间戳
